Remarks

Stores both the ushort destination archetype ID (backward-compat) and the direct Alis.Core.Ecs.Kernel.Archetypes.Archetype reference so the hot hit-path skips the WorldArchetypeTable lookup entirely. Round-robin (LRU-approx) eviction. Total size ~52 bytes (fits in one cache line).
